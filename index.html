DevVerse

DevVerse is an extension of github (consider it github++) for mobile that makes development easier. We focus on bringing important features missing in the Github Mobile app to users and to ensure that they can use github on the go with Twilio notications. We also ensure productivity on the go with custom project boards to keep track of their projects and assignments.

Features

  • Text message notifications on major Github Events (PR, Issue, Commit, Workflows/Actions)
  • Custom idea board to keep track of your projects and assignments
  • Repository management with the Github API

Tech Stack

The backend is a python fastAPI web api dockerirized with a postgres database to heroku cloud container service. The frontend is a SwiftUI iOS app built with Alamofire and Combine.
