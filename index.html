DevVerse

DevVerse is an extension of github (consider it Github++) for mobile that makes development easier.

Features

  • Text message notifications on major Github Events (PR, Issue, Commit, Workflows/Actions)
  • Custom idea board to keep track of your projects and assignments
  • Repository management with the Github API

Tech Stack

The backend is a python fastAPI web api dockerirized with a postgres database to heroku cloud container service. The frontend is a SwiftUI iOS app built with Alamofire and Combine. The backend uses the Twilio API to send text messages and the Github API to read information.
