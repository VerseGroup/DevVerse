DevVerse

DevVerse is an extension of Github (consider it Github++) for mobile that makes development easier on the go.

Features

  • Text message notifications (Twilio) on major Github Events (PR, Issue, Commit, Workflows/Actions)
  • Custom idea board to keep track of your projects and assignments from your phone
  • Repository management using the Github API

Tech Stack

The backend is a python fastAPI web api dockerized with a postgres database to a heroku cloud container service (Running Docker Compose). The frontend is a SwiftUI iOS app built with AlamoFire, Combine, and new Swift Async Await feature. The backend uses the Twilio API to send text messages and the Github API to read information and parse data to the iOS app.
