About DevVerse

Design Process of DevVerse

To make DevVerse we used a lot Github features to accelerate the development process. We used project boards within our organization to keep track of our indiviudal objectives and open them automatically as issues in their relevant repositories. This helped us communicated better as a team. We used Github pages to host this static product demonstration website made with simple html and css. We used Github Workflows/Actions to automatically lint/test and execute our backend code on commit and pull request. Beyond Github we used Adobe XD to prototype our models and the UI.
